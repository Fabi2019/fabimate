3.3 Resolver problemas con ecuaciones de segundo grado

Explicación

Una ecuación de segundo grado (o ecuación cuadrática) tiene la forma:

\( ax^2 + bx + c = 0 \), con \( a \neq 0 \)

Para resolver problemas, se deben seguir estos pasos:

  1. Leer el problema cuidadosamente.
  2. Plantear una ecuación cuadrática.
  3. Resolver la ecuación usando un método adecuado.
  4. Interpretar la solución en el contexto (descartar valores no válidos).

Métodos principales:

  • Factorización: cuando la ecuación se puede descomponer en dos factores.
  • Fórmula general:

\[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

  • Completar el cuadrado: método algebraico general.

Ejemplo

El producto de dos números consecutivos es 56. ¿Cuáles son esos números?

Planteamiento:

\[ x(x+1) = 56 \]

\[ x^2 + x - 56 = 0 \]

Factorizamos:

\[ (x+8)(x-7)=0 \]

Soluciones:

\( x = -8 \) o \( x = 7 \)

Respuesta: 7 y 8 (se descarta el negativo)

Ejercicios

1. Resolver: \( x^2 - 7x + 10 = 0 \)
2. Resolver: \( 3x^2 + x - 2 = 0 \)
3. El área de un rectángulo es 60 m². Un lado mide \( x \) y el otro \( x+5 \). Encuentra los lados.
4. Resolver usando fórmula general: \( x^2 + 2x - 3 = 0 \)
5. Un número multiplicado por sí mismo más 5 veces el número es 24. Encuentra el número.

Respuestas

  • 1. \( x=5, x=2 \)
  • 2. \( x=\frac{2}{3}, x=-1 \)
  • 3. \( 5 \) y \( 12 \)
  • 4. \( x=1, x=-3 \)
  • 5. \( x=3, x=-8 \) (válido según contexto)
