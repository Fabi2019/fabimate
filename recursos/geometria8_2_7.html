Elementos de una pirámide

Explicación

Una pirámide es un sólido geométrico con una base poligonal y caras triangulares que se unen en un punto llamado ápice. [1](https://www.mathsisfun.com/geometry/pyramids.html)

Dibujo

Ejemplo

Pirámide cuadrada con lado 6 cm y altura 4 cm.

Ejercicios

  1. Identifica la base en una pirámide triangular.
  2. ¿Cuántas caras laterales tiene una pirámide pentagonal?
  3. Diferencia entre altura y apotema.
  4. Dibuja una pirámide cuadrada.
  5. Si tiene 6 caras laterales, ¿qué base tiene?

Respuestas

  1. Triángulo.
  2. 5.
  3. Altura es perpendicular, apotema inclinada.
  4. Base cuadrada y caras triangulares.
  5. Hexágono.
