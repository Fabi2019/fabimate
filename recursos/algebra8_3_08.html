3.8 Suma, Resta y Multiplicación de Polinomios

Explicación

Un polinomio es una expresión algebraica formada por la suma o resta de varios monomios.

Operaciones con polinomios:
  • Suma: Se suman los términos semejantes (misma variable y exponente).
    \[ (3x^2 + 2x) + (5x^2 + x) = 8x^2 + 3x \]
  • Resta: Se cambian los signos del segundo polinomio y se suman términos semejantes.
    \[ (4x^2 + 3x) - (2x^2 + x) = 2x^2 + 2x \]
  • Multiplicación: Se multiplica cada término de un polinomio por cada término del otro.
    \[ (x + 2)(x + 3) = x^2 + 5x + 6 \]

Ejemplo

Operar:

\[ (2x^2 + 3x) + (x^2 + 5x) \]

Solución:

\[ 2x^2 + x^2 = 3x^2 \]

\[ 3x + 5x = 8x \]

Resultado:

\[ 3x^2 + 8x \]

Ejercicios

  1. \((x + 2) + (3x + 5)\)
  2. \((5x^2 + 2x) - (3x^2 + x)\)
  3. \((x + 4)(x + 2)\)
  4. \((2x)(3x^2 + 5)\)
  5. \((x^2 + x)(x)\)

Respuestas

  1. \(4x + 7\)
  2. \(2x^2 + x\)
  3. \(x^2 + 6x + 8\)
  4. \(6x^3 + 10x\)
  5. \(x^3 + x^2\)
