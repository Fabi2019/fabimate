3.6 Operaciones con Monomios

Explicación

Un monomio es una expresión algebraica formada por un número (coeficiente) y variables con exponentes enteros no negativos.

Ejemplo de monomio: \[ 3x^2,\quad -5ab,\quad 7y \]
Operaciones con monomios:
  • Suma y resta: Solo se pueden sumar o restar monomios semejantes (misma parte literal).
    \[ 3x + 5x = 8x \] \[ 7a^2 - 2a^2 = 5a^2 \]
  • Multiplicación: Se multiplican coeficientes y se suman exponentes.
    \[ (2x^2) \cdot (3x^3) = 6x^5 \]
  • División: Se dividen coeficientes y se restan exponentes.
    \[ \frac{6x^5}{2x^2} = 3x^3 \]

Ejemplo

Operar:

\[ (4x^2 + 3x^2) + (2x^2) \]

Solución:

\[ 4x^2 + 3x^2 = 7x^2 \]

\[ 7x^2 + 2x^2 = 9x^2 \]

Resultado: \(9x^2\)

Ejercicios

  1. \(3x + 7x\)
  2. \(10a^2 - 4a^2\)
  3. \((2x^3)(5x^2)\)
  4. \(\frac{12y^4}{3y}\)
  5. \(6m^2 + 3m\)

Respuestas

  1. \(10x\)
  2. \(6a^2\)
  3. \(10x^5\)
  4. \(4y^3\)
  5. No se pueden sumar (no son semejantes)
