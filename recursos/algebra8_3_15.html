3.15 Resolver ecuaciones literales para una de las letras

Explicación

Una ecuación literal es una ecuación que contiene dos o más letras. Resolver una ecuación literal consiste en despejar una de las letras para expresarla en función de las demás.

Pasos para resolver una ecuación literal

  1. Identificar la letra que se desea despejar.
  2. Aplicar operaciones equivalentes en ambos miembros.
  3. Mover los términos necesarios para dejar sola la letra.
  4. Simplificar la expresión obtenida.

Ejemplos de ecuaciones literales

\[ A=b+h \]

Despejar \(b\):

\[ A-h=b \]

Por lo tanto:

\[ b=A-h \]

Ejemplo

Despejar \(x\) en:

\[ a=x+b \]

Paso 1: Restar \(b\) en ambos miembros.

\[ a-b=x+b-b \] \[ a-b=x \]

Respuesta:

\[ x=a-b \]

Ejemplo Aplicado

Despejar \(r\) en la fórmula del perímetro de una circunferencia:

\[ P=2\pi r \]

Paso 1: Dividir ambos miembros entre \(2\pi\).

\[ \frac{P}{2\pi}=r \]

Resultado:

\[ r=\frac{P}{2\pi} \]

Ejercicios

  1. Despejar \(a\) en: \[ c=a+b \]
  2. Despejar \(x\) en: \[ m=3x \]
    3. en: \[ A=a+b \]
  3. Despejar \(r\) en: \[ A=\pi r^2 \]
  4. Despejar \(h\) en: \[ A=\frac{b\cdot h}{2} \]

Respuestas

  1. \[ a=c-b \]
  2. \[ x=\frac{m}{3} \]
  3. \[ b=A-a \]
  4. \[ r=\sqrt{\frac{A}{\pi}} \]
  5. \[ 2A=b\cdot h \] \[ h=\frac{2A}{b} \]
