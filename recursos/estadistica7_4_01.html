4.2 Tipos de Datos: Cuantitativos y Cualitativos

Explicación

Las variables estadísticas se clasifican según el tipo de información que representan:

Datos cualitativos:
  • Describen cualidades.
  • Nominal: sin orden (ej: color).
  • Ordinal: con orden (ej: nivel educativo).
Datos cuantitativos:
  • Se expresan con números.
  • Discreto: valores enteros (ej: número de hijos).
  • Continuo: valores reales (ej: altura).

Ejemplo

  • Sexo → Cualitativa nominal
  • Edad → Cuantitativa discreta
  • Altura → Cuantitativa continua
  • Satisfacción → Cualitativa ordinal

Ejercicios

  1. Número de mascotas
  2. Color favorito
  3. Temperatura corporal
  4. Nivel académico
  5. Cantidad de estudiantes

Respuestas

  1. Cuantitativo discreto
  2. Cualitativo nominal
  3. Cuantitativo continuo
  4. Cualitativo ordinal
  5. Cuantitativo discreto
