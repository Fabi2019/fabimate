Elementos Básicos de la Geometría

Explicación

En geometría usamos nomenclatura para nombrar correctamente las figuras.

Punto

   ● A

Se representa con una letra mayúscula.
Ejemplo: punto $A$

Recta

←──────────────→
      A   B

Se nombra con dos puntos o una letra.
Ejemplo: recta $AB$ o recta $l$

Segmento

A ●────────● B

Se escribe como:
$\overline{AB}$

Semirrecta (rayo)

A ●────────────→ B

Se escribe como:
$\overrightarrow{AB}$
El punto A es el origen

Plano

 -----------------------
|   A     B     C       |
|        PLANO α        |
|                       |
 -----------------------

Se nombra con tres puntos o una letra griega.
Ejemplo: plano $ABC$ o plano $\alpha$

Puntos colineales

A ●────●────● C
     B

Se escriben como:
Los puntos $A, B, C$ son colineales

Puntos no colineales

A ●
      ● B

   ● C

$A, B, C$ no son colineales

Puntos coplanares

Plano α
A ●     B ●

   C ●

$A, B, C$ pertenecen al plano $\alpha$

Puntos no coplanares

Plano α: A ●  B ●

Plano β: C ●

El punto $C$ no pertenece al plano $\alpha$

Ejemplo

A ●────● B ─────→

      ● C

$\overline{AB}$ es un segmento
$\overrightarrow{AB}$ representa una semirrecta
Los puntos $A$ y $B$ son colineales
El punto $C$ es no colineal

Ejercicios

1. Escribe la notación de un segmento entre A y B.

2. Escribe la notación de una semirrecta desde A hacia B.

3. Nombra una recta usando dos puntos.

4. Escribe cómo se representa un plano.

5. Indica si A, B, C son colineales o no en un dibujo.

Respuestas

1. $\overline{AB}$
2. $\overrightarrow{AB}$
3. $AB$
4. Plano $ABC$ o $\alpha$
5. Depende del dibujo
