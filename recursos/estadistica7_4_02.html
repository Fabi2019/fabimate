4.2 Tipos de Datos: Cuantitativos y Cualitativos

Explicación

En estadística, las variables se clasifican según el tipo de dato que representan:

1. Datos Cualitativos:
Describen cualidades o características que no se expresan con números.
  • Nominales: no tienen orden (ejemplo: color de ojos).
  • Ordinales: tienen un orden (ejemplo: nivel educativo).
2. Datos Cuantitativos:
Se representan mediante números y se pueden medir.
  • Discretos: valores enteros (ejemplo: número de hijos).
  • Continuos: valores con decimales (ejemplo: peso, altura).

Ejemplo de clasificación:

\[ \text{Edad} = 15 \quad \Rightarrow \quad \text{Cuantitativo} \]

\[ \text{Color} = \text{Azul} \quad \Rightarrow \quad \text{Cualitativo} \]

Ejemplo

Se analizan las siguientes variables en un grupo de personas:

  • Sexo
  • Edad
  • Altura
  • Nivel de satisfacción (bajo, medio, alto)

Clasificación:

  • Sexo → Cualitativa nominal
  • Edad → Cuantitativa discreta
  • Altura → Cuantitativa continua
  • Satisfacción → Cualitativa ordinal

Ejercicios

  1. Clasifica: número de mascotas en una casa.
  2. Clasifica: color favorito.
  3. Clasifica: temperatura corporal.
  4. Clasifica: nivel académico.
  5. Clasifica: cantidad de estudiantes en un aula.

Respuestas

  1. Cuantitativo discreto
  2. Cualitativo nominal
  3. Cuantitativo continuo
  4. Cualitativo ordinal
  5. Cuantitativo discreto
