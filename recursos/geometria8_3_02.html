3.2 Representación tabular, algebraica y gráfica de una función lineal

Explicación

Una función lineal puede representarse de tres formas:

  • Algebraica: mediante una ecuación $$y = ax + b$$
  • Tabular: mediante una tabla de valores
  • Gráfica: en el plano cartesiano como una recta

Una función lineal siempre forma una línea recta en el plano cartesiano [1](https://www.unprofesor.com/matematicas/que-es-una-funcion-lineal-ejemplos-6821.html)

Para representarla:

  1. Se eligen valores de $$x$$
  2. Se calculan los valores de $$y$$
  3. Se forman pares ordenados $$(x,y)$$
  4. Se grafican los puntos y se unen
---

Ejemplo

Representar la función:

$$ y = 2x + 1 $$

Gráfica

✅ La gráfica es una recta creciente porque $$a = 2 > 0$$

---

Ejercicios

1. Representar (tabla y gráfica) la función $$y = x + 2$$.
2. Construir una tabla para $$y = 3x$$.
3. Graficar $$y = -x + 1$$.
4. Completar la tabla:
x: 0,1,2
y: ?, ?, ? para $$y = 2x + 3$$
5. Identificar si la función $$y = -2x + 4$$ es creciente o decreciente.
---

Respuestas

1. Tabla y recta correspondiente.

2. y = 0, 3, 6...

3. Recta decreciente.

4. y = 3, 5, 7

5. Decreciente
